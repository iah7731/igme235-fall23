Original Design

References

Tutorials referenced include W3Schools' pages on CSS grid, horizontal navbars, selectors, text align, images, links and coloring them, semantic markup, buttons, borders, and Flexbox.

Other materials referenced included LogRocket's local storage tutorial and Coolors for the color palette inspiration. This project used the Jikan REST API v4.

Design Goal

I want to make an anime searcher using Jikan REST API v4. The purpose is to provide a simple way to search through all of the anime that exists. My audience is any anime watcher, but mainly the older demographic of that audience by making it simple and easy to use. The main functionality I want to include is searching for anime with a description and image alongside sorting features, including rating, air date, and genre.
